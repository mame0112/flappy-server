Overview

What's flappy?

flappy is loose communication tool that enable you to communicate with your friend without stress.

Stress-less messaging

Don\'t you think current messaging apps is stressful? Too many messages and unanswered messages etc. Yes? Then, it would be good to use this "Flappy". It enable you to communicate with your friend without stress by to be disappeared message.

To be disappeard message

One of characteristics feature of flappy is to be disappeard message. If you don\'t read new message more than about 30 minutes, it will disappear automatically and you will not be abou to read it anymore.\n\n Apparently, it seems inconvenient as a messaging service, but for the sake of it, you never.

So what?

This messagin service wiill not make message receiver reply message because you and your friend might missed the message before disappearing. It means message sender can send message without considering situation of receiver. As a result, you can send message more freely than now!
